## Dual Scoring System

Every extracted element receives two scores:

### Confidence Score (0.0 - 1.0)
"How sure am I this entity/relationship exists in the text?"

- 0.9+ = Explicitly stated
- 0.7-0.9 = Strongly implied
- 0.5-0.7 = Reasonably inferred
- Below 0.5 = Speculative

### Strength Score (0.0 - 1.0)
"How strongly does this node activate for this content?"

- 0.9+ = Dominant theme
- 0.7-0.9 = Strong presence
- 0.4-0.7 = Moderate presence
- Below 0.4 = Weak/background

**Example:** An entity might have:
- High confidence (0.95) - clearly mentioned
- Low ANCHOR strength (0.2) - doesn't ground to physical
- High GATEWAY strength (0.85) - strongly signals passage
